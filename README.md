# Sticks
Developed in 2023.

This bot was developed for an online community Discord server. The bot added the following features:
1. Interactive Picture Identification Game
2. User Profile and Economy System
3. Anonymous Message Submissions
4. Staff Content Management
5. Automated Server Banner Rotation
6. Reddit Integration
7. Command Handling (Slash Commands)
8. Database Integration (MongoDB)
9. Daily Reward System
10. Suggestion Submission System
11. Partnered Server Invite Commands
12. Channel Name Changing
13. Event Handling
14. Modal Interactions

Not included in this repo is the .env file containing private variables to connect with MongoDB and Discord's API.
